# 📝 Todo Application (React)

A simple and clean **Todo Application built with React.js** to manage daily tasks.

This project focuses on building something practical while understanding how React actually works — not just making things look good, but making them work smoothly.

---

## 🚀 What it does

This app lets you:

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Keep track of your daily work  

Simple, useful, and focused on real functionality.

---

## 💡 Why I built this

Instead of jumping into big projects, I focused on something small to strengthen my fundamentals.

While building this, I worked on:

- Managing state in React  
- Handling user input  
- Updating UI dynamically  
- Writing cleaner, more readable code  

It’s a simple project, but it helped me understand how React behaves in real scenarios.

---

## 🛠️ Tech Used

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3
